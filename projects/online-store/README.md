# Проект: Интернет-магазин электроники "ТехноСфера"

## О проекте

Это **учебный кейс**, созданный для демонстрации навыков 
системного аналитика.

Проект основан на анализе реального рынка электроники 
(конкуренты: DNS, Ситилинк) и типовых бизнес-процессов 
интернет-магазинов.

## Цель кейса

Показать умение:
- Проводить интервью с заказчиком
- Собирать и приоритизировать требования
- Документировать результаты
- Создавать User Stories и диаграммы
- Структурировать аналитическую работу
